# ChatGovorun

Telegram chatbot integrated with YandexGPT.

## Description
This project implements a Telegram bot that uses YandexGPT API for generating responses. The bot is deployed on Amvera hosting platform.

## Technologies
- Python
- Telegram Bot API
- YandexGPT API
- Amvera hosting
